# Certora Prover Documentation

Certora Prover is a formal verification system for smart contracts. The public documentation is organized around the Certora User's Guide, the Certora Verification Language reference, and the Certora Prover reference.

For EVM developers, the key workflow is to write CVL specifications that describe contract properties, then run the prover against Solidity contracts to check whether those properties can be violated. The documentation also links to prover tutorials, CVL examples, and related products such as the equivalence checker and mutation-testing tools.

Developer relevance in this graph: use Certora-style formal specifications when a protocol requires machine-checked guarantees for access control, accounting invariants, authorization boundaries, or equivalence across implementations.
